💡 LumiControl

Sistema de monitoramento e controle de iluminação inteligente com IoT.

🚀 Tecnologias

- Node.js
- Express
- HTML5 semântico
- CSS3
- JavaScript

 ⚙️ Como rodar
```bash
npm install
node server.js
```

Acesse: http://localhost:3000

📡 Rotas da API

| Método | Rota | Descrição |
|--------|------|-----------|
| GET | /salas | Lista todas as salas |
| PUT | /salas/:id | Liga ou desliga uma sala |
| GET | /sensores | Lista os sensores |
| GET | /energia | Dados de energia |
